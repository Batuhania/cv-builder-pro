# CV Builder Pro

A professional, client-side CV/Resume builder built with pure HTML, CSS, and JavaScript.

![Preview](preview.png)

## Features

- **Real-time Editing**: Click and edit any text on the CV.
- **Drag & Drop**: Reorder sections, skills, and experience items easily.
- **Multiple Themes**: Choose from Classic, Modern, Creative, and Professional designs.
- **Photo Upload**: Client-side photo cropping and compression.
- **PDF Export**: High-quality A4 PDF export.
- **Local Storage**: Auto-save progress to your browser.
- **PWA Support**: Installable as a progressive web app.
- **Multi-language**: Support for English, Turkish, German, Spanish, and French.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Start-Of-The-Art/devggam.git
   ```
2. Open `index.html` in your browser.

## Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern variables/properties, Flexbox/Grid
- **JavaScript**: ES6+ modules
